# Microservices Incident Center

## Стек технологий
* **DB:** SQLite
* **Инструменты:** Pycharm, Git

## Структура базы данных
в проекте реализованы:
1. **Таблица `teams`**: Список команд и их техстек.
2. **Таблица `services`**: Список микросервисов со статусами (`stable`, `down`).
3. **Relationship**: Связь `One-to-Many` через `FOREIGN KEY`.

## Как запустить
1. Склонируйте репозиторий.
2. откройте `init.sql` в любом SQL-клиенте (или PyCharm).
3. Запустите скрипт, чтобы создать таблицы и наполнить их тестовыми данными.

## Полезные запросы
в базе создана **VIEW** `v_problem_services`, которая показывает только упавшие сервисы.
